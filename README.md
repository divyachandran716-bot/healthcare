🏥 Healthcare Management Dashboard
📌 About the Project

The Healthcare Management Dashboard is a React-based web application designed to manage hospital operations efficiently. It provides a centralized system for handling patients, doctors, nurses, appointments, medicines, pharmacy, and schedules.

This project simulates a real-world hospital management system with a clean UI and role-based dashboards for different users such as Admin, Doctor, and Nurse.

🚀 Features
👨‍⚕️ Admin, Doctor, and Nurse dashboards
🏥 Patient management system
📅 Appointment scheduling system
💊 Medicines and pharmacy stock management
📊 Dashboard analytics UI
🧾 Patient admission, ICU, and discharge pages
🔐 Authentication context (login/logout system)
🧩 Reusable React components
📱 Responsive layout (basic structure)
🛠️ Tech Stack

React.js
React Router DOM
JavaScript (ES6+)
HTML5 & CSS3
Vite (build tool)

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/healthcare-dashboard.git
2. Move into project folder
cd healthcare-dashboard
3. Install dependencies
npm install
4. Run the project
npm run dev
